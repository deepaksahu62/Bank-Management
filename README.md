# 🏦 Bank Management System (Streamlit)

A simple **Bank Management System** built using **Python + Streamlit**.
This app allows users to create accounts, deposit/withdraw money, update details, and delete accounts with persistent storage using JSON.

---

## 🚀 Features

* ✅ Create new bank account
* 💰 Deposit money
* 💸 Withdraw money
* 📄 View account details
* ✏️ Update user details
* ❌ Delete account
* 🔄 Auto data migration (old data support)
* 💾 JSON-based database

---

## 🛠️ Tech Stack

* Python
* Streamlit
* JSON (for data storage)

---

## ⚠️ Validations

* Age must be **18+**
* PIN must be **4 digits**
* Deposit limit: **1 – 10000**
* Withdrawal only if sufficient balance
